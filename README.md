# Wangzi Skills

Agent skill collection. Each skill lives under `skills/<name>/` with a `SKILL.md` entry point.

**Keywords:** agent skills, creator workflow, knowledge workflow.

## Install

```bash
npx skills add cwandev/wangzi-skills
```

## Skills

| Skill | Link |
|-------|------|
| wangzi-excalidraw-storyboard | [`skills/wangzi-excalidraw-storyboard/SKILL.md`](skills/wangzi-excalidraw-storyboard/SKILL.md) |

Add new rows here when you add directories under `skills/`.

## Layout

```text
skills/<skill-name>/SKILL.md
skills/<skill-name>/references/   # optional
skills/<skill-name>/evals/         # optional
```
