# Diagram modes (heuristics)

Use this table as a **guide** when choosing sketch type; explicit user instructions take priority.

| Content signal | Sketch bias |
|----------------|-------------|
| Multi-step flow, branches, loops, state changes | Flowchart, decision diamonds, lanes/phases, simple state sketch |
| Many entities and relations (depends on, drives, blocks) | Relationship map, arrow network, groups + labels |
| Parallel points, light structure, “three things” talks | Numbered/bulleted blocks, columns, light grouping |
| Single idea, metaphor, vibe, “just illustrate it” | Quick hand-drawn feel, big icon + short caption, scene-style doodle |
| Compare before/after, pros/cons | Side-by-side or top-bottom split |
| Timeline, evolution | Horizontal or vertical timeline |

**Combining is normal**—for example flow plus numbered steps, or a relationship map plus a headline title. Optimize for **clear to say, clear to see, not crowded**.
