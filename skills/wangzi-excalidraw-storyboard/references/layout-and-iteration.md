# Layout defaults and iteration

## Canvas defaults (vertical talking-head; not mandatory)

For **vertical video + bottom-right face cam**:

- **Suggest** portrait aspect (~3:4, e.g. `600×800`).
- **Suggest** a bottom-right picture-in-picture–safe band; keep key content out of that region.
- If landscape, full-bleed, or minimal center composition fits the topic or user request better, **that is fine**—do not force a single template.

## Iteration

When the user gives feedback:

1. Honor **spacing, proportion, and style** first.
2. If diagram content drifts from the source, reconcile with **fidelity** to the input material (no invented facts).
